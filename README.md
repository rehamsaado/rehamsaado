### 📁 Featured Projects

#### 🏫 Smart School (Graduation Project)
A comprehensive, enterprise-grade school management and communication ecosystem designed to bridge the gap between students and teachers.
* **Architecture:** Strictly adhered to **Clean Architecture** by decoupling business logic from UI across three distinct layers: *Data*, *Domain*, and *Presentation*.
* **State Management:** Powered by **BLoC/Cubit** to achieve predictable state changes and efficient data flow.
* **Networking:** Integrated robust REST APIs via **Dio** for dynamic data synchronization and precise JSON parsing.
* **UI/UX:** Built highly responsive layouts for diverse screen sizes with fully reusable widgets, academic grids, and interactive promotional carousels.
* 🔗 [**View Code Repository**](https://github.com/rehamsaado)

#### 💬 Swift Chat (Real-time Messenger)
A full-featured instant messaging application engineered from scratch to ensure extreme speed and ultra-low latency.
* **Real-time Engine:** Fully integrated **Supabase Streams** and PostgreSQL backends leveraging WebSockets for reactive message delivery.
* **State & Logic:** Orchestrated using the **BLoC pattern** within a structured Clean Architecture pipeline to handle asynchronous data streams natively.
* **UI Components:** Designed pixel-perfect, flexible chat interfaces and modular custom widgets built for scale.
* ⚙️ **DevOps Integration:** Automated quality checks and testing pipelines using **GitHub Actions (CI/CD)**.
